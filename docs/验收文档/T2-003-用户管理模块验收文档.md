### 2. API接口实现

| API路径 | 方法 | 功能描述 | 状态 |
|---------|------|----------|------|
| `/api/users/me` | GET | 获取当前登录用户的详细信息 | ✅ 完成 |
| `/api/users/me` | PUT | 更新当前用户信息 | ✅ 完成 |
| `/api/users` | GET | 获取用户列表（仅管理员） | ✅ 完成 |
| `/api/users/{user_id}` | GET | 获取指定用户的详细信息（仅管理员） | ✅ 完成 |
| `/api/users/{user_id}/role` | PUT | 修改用户的角色和管理员权限（仅管理员） | ✅ 完成 |
| `/api/users/{user_id}` | DELETE | 删除指定用户（仅管理员） | ✅ 完成 |
| `/api/users/batch` | POST | 批量添加教师用户（仅管理员） | ✅ 完成 |
